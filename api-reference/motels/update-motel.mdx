---
title: "PUT /api/motels/{id}"
description: "Update an existing motel. Admins can update any motel; property owners can only update their own."
---

## Overview

Updates the details of an existing motel. Authorization rules:

- **ADMIN** — can update any motel without restriction.
- **PROPERTY_OWNER** — can only update motels where `propertyId` matches their own user ID.

All fields in the request body are applied to the motel. The `approvalStatus` and `propertyId` are preserved from the existing record and cannot be changed through this endpoint.

## Request

**Method:** PUT  
**Path:** `/api/motels/{id}`  
**Auth required:** Yes (Bearer token)  
**Required role:** PROPERTY_OWNER (own motels only) or ADMIN (any motel)

### Path parameters

<ParamField path="id" type="integer" required>
  Numeric ID of the motel to update.
</ParamField>

### Request body

<ParamField body="name" type="string" required>
  Updated name. Between 3 and 100 characters.
</ParamField>

<ParamField body="address" type="string" required>
  Updated street address. Maximum 255 characters.
</ParamField>

<ParamField body="city" type="string" required>
  Updated city. Maximum 100 characters.
</ParamField>

<ParamField body="phoneNumber" type="string">
  Updated phone number. Maximum 20 characters.
</ParamField>

<ParamField body="description" type="string">
  Updated description. Maximum 500 characters.
</ParamField>

<ParamField body="imageUrls" type="array">
  Updated list of image URLs. Maximum 10 items; each URL up to 500 characters.
</ParamField>

<ParamField body="latitude" type="number">
  Updated geographic latitude. Must be between -90 and 90.
</ParamField>

<ParamField body="longitude" type="number">
  Updated geographic longitude. Must be between -180 and 180.
</ParamField>

<ParamField body="rues" type="string">
  Updated RUES number. Maximum 100 characters.
</ParamField>

<ParamField body="rnt" type="string">
  Updated RNT number. Maximum 100 characters.
</ParamField>

<ParamField body="ownerDocumentType" type="string">
  Updated owner document type. One of: `CC`, `NIT`, `CE`, `PASAPORTE`.
</ParamField>

<ParamField body="ownerDocumentNumber" type="string">
  Updated owner document number. Maximum 50 characters.
</ParamField>

<ParamField body="ownerFullName" type="string">
  Updated full legal name of the owner. Maximum 200 characters.
</ParamField>

<ParamField body="legalRepresentativeName" type="string">
  Updated legal representative name. Maximum 200 characters.
</ParamField>

<ParamField body="legalDocumentUrl" type="string">
  Updated URL of the primary legal document. Maximum 500 characters.
</ParamField>


## Error responses

| Status | Description |
|--------|-------------|
| 400 | Validation error — `name`, `address`, or `city` is blank, or a constraint is violated |
| 401 | Missing or invalid JWT, or `X-User-Username` header is absent |
| 403 | PROPERTY_OWNER attempting to update a motel they do not own, or user lacks required role |
| 404 | Motel not found, or the authenticated username was not found in the user database |
